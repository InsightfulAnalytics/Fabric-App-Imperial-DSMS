Square icon-only control for toolbars, table rows, and the collapsed rail. The `active` state shows the imperial wash + accent stripe.

```jsx
<IconButton icon="settings" title="Settings" />
<IconButton icon="layout-dashboard" title="Briefing" active />
<IconButton icon="more-horizontal" variant="solid" title="Actions" />
```

Props: `icon` (Lucide), `size` (`sm`/`md`), `variant` (`ghost`/`solid`), `active`, `disabled`, `title` (label + tooltip).
