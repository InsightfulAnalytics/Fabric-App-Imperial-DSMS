Primary action control — ALL-CAPS Rajdhani label, 2px radius, flat fills. Use `primary`/`danger` (imperial red) sparingly for the single most important action.

```jsx
<Button variant="primary" icon="play">Authorize</Button>
<Button>Cancel</Button>
<Button variant="ghost" icon="download" size="sm">Export</Button>
```

Variants: `primary`, `danger` (both imperial red), `secondary` (bay surface + hairline), `ghost` (transparent until hover). Sizes: `sm` (28px), `md` (34px). Props: `icon`, `iconTrailing` (Lucide names), `disabled`, `block`.
